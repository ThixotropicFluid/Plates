# Plates

Plates is a libray to create more realistic terrain using ideas from techtonic plates

## Running

To run the demo

```
git clone https://github.com/ThixotropicFluid/Plates
cd Plates
cmake 
cmake --build .
```
an executable will be in Debug

Controlls:

W - Forawrd

A - Left Strafe

S - Back

D - Right Strafe

Z - Focus 0, 0, 0

C - Cycle Terrain Texture
